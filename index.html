Modern Form Styles

By Levi Morrison

This page illustrates how simple form markup can be used to create beautiful styles that are also mobile compatible. The classes in the package are defined below. No JavaScript is used for styling. It requires the meyerweb reset.

ui-form
Apply to the form elements that need styling.
blue, red
Apply to the button elements to change their color.
required
Apply to labels/legends for the fields/fieldsets that are required. It will automatically add .
error
Apply to inputs and textareas when there is an error with the field. Also apply to the labels and legends for erroneous fields and fieldsets.

No data is collected on any of these forms.
